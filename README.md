# Controle de Gastos Pessoais com Firebase

Este é um aplicativo Android desenvolvido em Kotlin para a gestão e controle de gastos financeiros em tempo real.

## 🚀 Funcionalidades
* Cadastro de novos gastos com descrição e valor.
* Sincronização em tempo real com o banco de dados Firebase Realtime Database.
* Cálculo automático do saldo total acumulado diretamente na Dashboard.

## 🛠️ Tecnologias Utilizadas
* **Linguagem:** Kotlin
* **Banco de Dados:** Firebase Realtime Database
* **IDE:** Android Studio
